Building

The current version of the library can be found on the Mons Pico site. Go ahead check out the library with:

git clone https://github.com/mons-pico/pico.git

The CPico library is built using CMake. You'll need to have CMake installed to build the library, and you will also (obviously) need a C compiler.

It is recommended that you build "out of source." Do the following from the top-level folder.

mkdir build
cd build
cmake ..
make

At the end of this you should have the library and an executable named pico. All the build products are now in the build folder, which you can easily delete to clean up.
